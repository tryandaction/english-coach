# English Coach 使用指南

这份说明面向当前本地打包出的 **Open Source / 开源版**。

## 1. 建议你怎么测

### 先测 coach 主路径

优先按这个顺序体验：

1. Setup
2. Home
3. Vocabulary / Grammar
4. Reading / Listening
5. Progress / History

这样最容易判断产品是不是已经具备“监督者 / 帮助者 / 教练”的基本形态。

### 然后再测 AI 路径

如果你配置了自己的 API Key，再继续测：

1. Writing
2. Speaking
3. Chat

## 2. 没有 API Key 时

当前仍可测试：

- Vocabulary / Word Books / SRS
- Grammar
- Reading 离线训练
- Listening 内置脚本流程
- Practice 页面中的离线可练部分
- Mock Exam section flow
- Home / Progress / History
- 每日计划、测试提醒与本地复盘

## 3. 有 API Key 时

你可以继续测试：

- Chat 是否能进入
- Writing 是否能拿到反馈
- Speaking 是否能拿到评分
- Writing / Speaking 的任务是否尽量不重复
- Home / Setup / Progress 的 AI 状态是否一致

## 4. 本轮重点观察什么

- 首页是否能告诉用户今天做什么
- 是否能一键进入真实训练页，而不是多层跳转
- 无 AI 时是否仍能顺畅使用离线主流程
- 完成训练后，用户是否能看到结果、复盘和下一步动作
- 提醒是否有用，但不过度打扰

## 5. 当前限制

- `Mock Exam` 不是标准化统分系统
- `Writing / Speaking` 的高质量反馈依赖 API Key
- 当前内容质量仍在持续优化，不应视作最终定版
